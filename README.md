### Projects

- [use-watcher-map](https://github.com/lachlanmcmillan/use-watcher-map) — Control re-renders in React with path-based subscriptions, so components only re-render when relevant data changes. [Live demo](https://lachlanmcmillan.github.io/use-watcher-map/)
- [use-watcher-form](https://github.com/lachlanmcmillan/use-watcher-form) — High-performance React form library where inputs are uncontrolled by default — typing never causes re-renders. Built on use-watcher-map. [Live examples](https://lachlanmcmillan.github.io/use-watcher-form/)
- [better-git-branch](https://github.com/lachlanmcmillan/better-git-branch) — Quickly change branches in your repo.
